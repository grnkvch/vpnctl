# Project

Goal: manage a personal VPN infrastructure.

Assumptions:
- VPS already exists
- SSH access already exists
- Tool starts from an SSH-accessible server

VPN backend:
- WireGuard

Routing layer:
- Mihomo / Clash Mi

Supported clients:
- iPhone
- macOS
- Arch Linux
- Ubuntu
- Linux VM

Scale:
- 1 operator
- 1-20 clients
- simplicity preferred over flexibility
