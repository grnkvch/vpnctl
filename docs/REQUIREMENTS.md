# Requirements

## Client Lifecycle

- create
- revoke
- rotate-keys
- regenerate-config
- delete

## Config Types

- WireGuard
- Clash/Mihomo

## Config Delivery

Evaluate:

- QR codes
- local files
- temporary links
- signed URLs
- one-time links

## Security

- Never log private keys
- Minimize secret exposure
- SSH-based execution

## ADR Requirement

Maintain Architecture Decision Records in docs/ADR.

Each ADR should contain:

- Context
- Decision
- Alternatives Considered
- Tradeoffs
- Consequences
